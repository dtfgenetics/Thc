# Feature Roadmap

This roadmap defines what the game should become and what Codex should build first.

## Confirmed player count

Support up to 10 players.

- Local pass-and-play: 2-10 players.
- Online multiplayer: 2-10 players.
- Default simple testing: 2-4 players.
- Stress testing: 5-10 players.

## Must-have first playable features

- 2-10 player setup.
- Dice roll.
- Turn indicator.
- Space-by-space token movement.
- Action-card spaces.
- Forward movement cards.
- Backward movement cards.
- Skip-turn effects.
- Roll-again effect.
- Win condition.
- Restart game.
- Mute/unmute.
- Mobile responsive layout.

## 10-player required UI changes

- Compact player list.
- Current player highlight.
- Token labels 1-10.
- Token spread around shared spaces.
- Faster animation option.
- Optional turn timer later for online rooms.

## Board feature options

- Card space: draw action card.
- Skip space: skip next turn.
- Boost space: move ahead 2.
- Trap space: move back 2.
- Swap space: swap with the leader.
- Safe space: ignore the next bad card.
- Shortcut space: jump to a later space.
- Slide-back space: move backward to a marked space.

## Card deck expansion

Prototype deck: 10 cards.
First public deck: 30 cards.
Expanded deck: 60 cards.

Card categories:

- Move forward.
- Move backward.
- Skip turn.
- Roll again.
- Swap positions.
- Jump to named zone.
- Protect yourself.
- Cancel a bad card.
- Pull another player back.
- Push the last-place player forward.

## Audio and visual polish

- Background music loop.
- Dice roll sound.
- Token move tick sound.
- Card draw sound.
- Good-card sound.
- Bad-card sound.
- Skip sound.
- Victory sound.
- Loading screen.
- Animated card reveal.
- Winner celebration.

## dtfseeds.com features

- Static game deployed to `/games/high-land/`.
- WordPress page or menu link to the game.
- Game hub page later.
- Rules page.
- Share button.
- Version number shown in footer.
- Bug report or feedback link.

## Multiplayer features later

- Create room.
- Join room code.
- Player name entry.
- Token/color choice.
- Ready button.
- Host starts game.
- Server-owned dice rolls.
- Server-owned card deck.
- Server-owned turn order.
- Reconnect support.
- Turn timer.
- Idle auto-skip.

## Not yet

Do not add paid accounts, gambling, real prizes, crypto, or complicated economies. Finish the playable board game first.
