# Playtest Matrix

## Required manual tests

| Test | Players | Goal |
|---|---:|---|
| Small game | 2 | Finish one full game. |
| Standard game | 4 | Check turn order and card pacing. |
| Large game | 10 | Check token spread and player list. |
| Mobile game | 4 | Check layout on phone width. |
| Save load | 4 | Save, refresh, load, continue. |
| Card chain | 2 | Confirm draw-again resolves. |
| Group move | 2 | Confirm another player can win from a card. |

## Pass condition

No console errors, no stuck turns, and no invisible tokens.
