# Asset Import Checklist

## Board

- Add `apps/high-land-web/public/assets/images/board/high-land-board.png`.
- Run the app.
- Align every boardPath coordinate to the visible space center.

## Cards

- Add card images to `apps/high-land-web/public/assets/images/cards/`.
- Match each card image to an effect in `actionCards.ts`.
- Keep text readable on mobile.

## Audio

- Add audio files to `apps/high-land-web/public/assets/audio/`.
- Replace generated tones after files exist.
- Keep mute working.

## Tokens

- Add token images after local player markers are verified.
- Keep token numbers visible for 10 players.
