# Feature Options for 10 Player Mode

This is the feature list for making a 10-player version feel good instead of slow.

## Game setup options

- Player count: 2, 3, 4, 5, 6, 8, or 10.
- Player names.
- Token selection.
- Color selection.
- Random turn order.
- Manual turn order later.
- Fast mode toggle.
- Family-friendly/adult-theme copy toggle only if needed later.

## Turn pacing options

For 10 players, pacing matters.

Add:

- Fast animations.
- Auto-advance after card reveal.
- Skip animation button.
- Online turn timer later.
- Idle auto-skip for online mode later.

## Board options

- Normal spaces.
- Action card spaces.
- Skip spaces.
- Boost spaces.
- Trap spaces.
- Swap spaces.
- Shortcut spaces.
- Safe spaces.
- Zone gates that trigger unique effects.

## Card options

- Move forward.
- Move backward.
- Skip next turn.
- Roll again.
- Swap with leader.
- Swap with last place.
- Pull leader back.
- Boost last place.
- Protect from next bad card.
- Cancel a card.
- Jump to zone.
- Return to previous zone.

## Multiplayer room options later

- Create room.
- Join room code.
- Host controls.
- Ready button.
- Reconnect support.
- Spectator mode.
- Emote reactions.
- Simple room log.
- Turn timer.
- Host kick inactive player.

## Website options for dtfseeds.com

- Game hub page.
- High Land landing page.
- Rules page.
- Play now button.
- Share link.
- Feedback button.
- Mobile-first layout.
- Version number.
- Update notes.

## Do first

Build the 10-player local version first. Online multiplayer should use the same rules after local mode works.
