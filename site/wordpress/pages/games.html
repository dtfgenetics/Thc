What the status labels mean

  • Playable build: a public game can be opened and tested now.
  • Playable development build: core play exists, but the game is still being tested and improved.
  • In development: rules, content, artwork, print files, or code are still being produced.
  • Physical release: a game is not for sale until an individual product page publishes final components, price, policies, and availability.

Help improve the games

Useful playtest reports include the game name, device and browser, player count, exact steps that caused the problem, what was expected, what happened, and a screenshot when possible.
