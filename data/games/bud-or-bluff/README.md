# Bud or Bluff — Bluff Name Dataset

This directory contains fictional cannabis strain names for the **Bud or Bluff** game.

## Current dataset

- 200 unique bluff names
- 100 names from the original batch
- 100 names from the July 17, 2026 expansion
- JSON source: `bluff-names.json`
- Review database: [Bud or Bluff — 200 Screened Bluff Names](https://docs.google.com/spreadsheets/d/1RWTIZbFTjU7YC9IZtHkvnpCMqAwUAH5vvw2qmwbLoSE/edit)

## Verification status

Each full proposed name was screened for an exact collision against the working strain index and major public strain registries used by the project. No exact full-name match was found as of **2026-07-17**.

This status is not a guarantee that an unindexed local breeder has never used a name. Strain catalogs change, so every name must receive a fresh exact-name search and trademark review before commercial print or digital release.

## Production rule

The Google Sheet is the editorial source of truth for card IDs, categories, difficulty, screening status, and verification date. The JSON file is the lightweight repo source for game-development work.
