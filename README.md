# Browser Game Project

This repo contains the browser-based board game build.

## Run locally

```bash
cd apps/high-land-web
npm install
npm run dev
```

## Build

```bash
cd apps/high-land-web
npm run build
```

The production build outputs to:

```txt
apps/high-land-web/dist
```

## Current phase

Phase 1 is the local pass-and-play prototype:

- 2-4 local players
- dice roll
- turn order
- animated board movement
- action cards
- skip turns
- win condition
- simple placeholder board
- tests for core rules
